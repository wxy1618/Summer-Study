## 运行顺序

1. `01 colab_tutorial_homework.py`  
   对应 Colab Tutorial P10-P12、P22、P28 的基础练习，可到 Colab 运行，也可以在本地按 Python 脚本运行。

2. `02 environment_check.py`  
   用于检查当前 Python、WSL、PyTorch、Jupyter 环境状态。

3. `03 pytorch_intro_training.py`  
   PyTorch 入门训练代码，使用 CPU 即可运行。

4. `04 jupyter_notebook_practice.py`  
   Jupyter Notebook 单元格风格练习，适合在 VS Code 或 Jupyter 中按 `# %%` 单元格运行。

5. `05 paddle_pytorch_mapping.py`  
   PyTorch 与 PaddlePaddle 写法对照。若未安装 PaddlePaddle，脚本会跳过 Paddle 部分。
